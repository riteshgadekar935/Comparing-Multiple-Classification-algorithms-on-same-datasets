# Machine Learning Project | Nurse Attrition Prediction using Classification Algorithms

I'm excited to share my latest Data Science project: **Nurse Attrition Prediction**, where I built and compared multiple machine learning models to predict whether a nurse is likely to leave the organization.

# Project Objective

Employee attrition is a major challenge in the healthcare industry. The goal of this project was to analyze employee-related factors and build a predictive model that can help hospitals identify nurses at risk of leaving.

# Project Workflow

✅ Data Cleaning & Preprocessing
✅ Exploratory Data Analysis (EDA)
✅ Feature Engineering
✅ Correlation Analysis & Feature Selection
✅ Model Training & Evaluation
✅ Performance Comparison of Multiple Classification Algorithms

# Models Implemented

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Naive Bayes
* XGBoost
* Stacking Classifier

# Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC Score
* Confusion Matrix

# Key Learning

Instead of selecting a model based only on accuracy, I compared multiple evaluation metrics to choose the most appropriate model. This project also helped me understand the impact of class imbalance, feature correlation, preprocessing techniques, and model evaluation in real-world machine learning problems.
